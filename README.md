## Directions:
- install yay using 
```
sudo pacman -S paru
```

- install all packages with the following
```
paru -S fish kitty wofi neofetch quickshell spotify spicetify-cli hyprland hyprpaper hyprlock waybar starship mako rofi zsh grim slurp firefox-developer-edition zen-browser-bin zed visual-studio-code-bin gitkraken pass git-credential-manager docker docker-compose podman podman-compose virt-manager emacs python bun npm pros-cli gcc dotnet-runtime rustup
```

download the source files, and run the installation script
